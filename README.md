# 📘 TypeScript Learning Repository

This repository contains my journey of learning TypeScript from basic to advanced concepts. It includes examples, practice exercises, and mini projects to build strong TypeScript skills.

---

## 🚀 About TypeScript

TypeScript is a strongly typed superset of JavaScript that compiles into plain JavaScript. It helps developers write safer, scalable, and maintainable code.

## ⚙️ Setup Instructions

### 1. Install Node.js  
Download and install Node.js from official website.

### 2. Install TypeScript
```bash
npm install -g typescript

### 3. Verify Installation
```bash
tsc -v

### 4. Initialize Project
```bash
tsc --init

### 5. How to Run
```bash
tsc index.ts
node index.js
